## Technical Core Skills & Implementation

### 1. Advanced Data Transformation & ETL (Power Query)
* **Data Cleansing:** Standardized inconsistent, unstructured corporate benefit fields, platform titles, and localization attributes.
* **Feature Engineering:** Extracted boolean states from text elements to isolate critical job variables like WFH Availability, Degree Requirements, and Health Insurance Providing.
* **Performance Optimization:** Structured data types and eliminated redundant source schemas to keep model computation responsive.

### 2. Analytical Modeling & DAX (Data Analysis Expressions)
* **Robust Metrics:** Implemented custom statistical indicators bypassing basic arithmetic averages to calculate true Median Yearly Salary (113K USD) and Median Hourly Salary (47.62 USD).
* **Conditional Star-Rating System:** Built a dynamic, rule-based DAX rating system to visually score industry salary health directly via KPI cards.
* **Proportional Scaling:** Created calculated measures to dynamically compute percentages for organizational offerings for remote infrastructure and health coverage.

### 3. Data Architecture & Interactivity
* **Drill-Through Configuration:** Engineered an asynchronous page filtering mechanism allowing end-users to right-click specific roles and seamlessly slide into specialized granular breakdowns.
* **Frictionless UI Navigation:** Provisioned an explicit manual reset action via custom back-button mapping for fluid analytical iteration.

### 4. Advanced Data Visualization Selection
* **Trend Mapping:** Integrated micro Sparklines directly inside detailed matrix tables to map discrete, high-density timeline movements at a glance.
* **Correlation Mapping:** Leveraged a Scatter Plot layout to map the interaction between hourly rates and yearly salaries, allowing quick identification of financial outliers.
* **Geospatial Analytics:** Bound geolocation datasets directly to automated map visual layers for interactive global job-density mapping.
